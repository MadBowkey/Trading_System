# Audit Structure

Diese modulare Audit-Struktur ist verbindlich.

Verbindliche operative Audit-Dateien sind:
- `audit/modules/module_01.md` bis `audit/modules/module_10.md`
- `audit/conflict_register.md`
- `audit/decision_matrix.md`
- `audit/action_list.md`

`AUDIT.md` dient nur als ergänzende projektweite Übersicht und ersetzt die modulare Struktur nicht.

Keine Strukturänderungen während der laufenden Audit-Phase.
